============

Django Auth in RESTful way

### Installation

```console
git clone https://github.com/sunliwen/django-authr.git
mkvirtualenv django-authr
cd django-authr
make develop
```

### Run the tests

```console
make
```

### Fire up a temp server

```console
make run
```
